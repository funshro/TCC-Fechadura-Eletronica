Informações sobre a Fechadura Inteligente




Este projeto apresenta o desenvolvimento de uma fechadura inteligente avançada, capaz de controlar o acesso a áreas restritas de forma segura e eficiente. A fechadura utiliza tecnologias modernas como RFID para identificar os usuários, além de registrar todas as atividades e enviar os dados para um servidor remoto, permitindo um monitoramento completo do sistema. Em resumo, este projeto oferece uma solução completa e flexível para o controle de acesso, combinando segurança, praticidade e facilidade de uso.
Resumo

Resumo do Projeto: Fechadura Eletrônica Inteligente

A aplicação dessa tecnologia na porta das salas de aula visa não apenas facilitar o acesso dos professores. Além disso, a tecnologia RFID, proporciona um controle de acesso mais eficiente e seguro, eliminando a necessidade de múltiplas chaves físicas, mas também aumentar a segurança e a organização dentro da escola. Com a fechadura inteligente, os professores não precisarão mais carregar um grande número de chaves, o que simplifica sua rotina diária e reduz o risco de perda ou extravio de chaves e consequentemente melhora a segurança e a gestão dos espaços educacionais, criando um ambiente mais eficiente e seguro para todos os envolvidos
Funcionalidades

Principais funcionalidades:

Autenticação: Leitura de cartões RFID para identificar os usuários autorizados.
Controle de Acesso: Definição de diferentes níveis de acesso para cada usuário, permitindo um controle preciso sobre quem pode entrar em cada área.
Registro de Dados: Armazenamento de todas as informações sobre os acessos, como data, hora e usuário, em um banco de dados.
Interface Web: Possibilidade de gerenciar os usuários, visualizar o histórico de acessos e configurar o sistema através de uma interface web intuitiva.
Componentes

Equipamentos/ferramentas utilizadas

Hardware: Microcontrolador, leitor de cartões RFID, display (opcional), conexão Wi-Fi ou Ethernet.
Software: Microcontrolador com sistema operacional e linguagem de programação específicos, servidor com linguagem e framework adequados, banco de dados para armazenar as informações.
Funcionamento

Como a fechadura funciona?

Instalação: Montagem do hardware, configuração do software e conexão à rede.
Cadastro: Cadastro de usuários e definição de seus níveis de acesso através da interface web.
Utilização: Aproximação do cartão RFID para desbloquear a fechadura.
Futuro

Futuras melhorias

Notificações por email ou SMS em caso de eventos como acessos não autorizados.
Integração com sistemas de controle de acesso mais complexos.
Adicionar suporte a outros métodos de autenticação, como reconhecimento facial.
